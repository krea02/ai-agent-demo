# Kraja / vandalizem – posebnosti
Pri kraji ali vlomu je obvezno:
- takojšnja prijava policiji.

Stranka naj pripravi:
- policijski zapisnik / številko prijave,
- fotografije škode in okolice,
- seznam ukradenih predmetov (če relevantno).

Prijava zavarovalnici:
- preko spletnega obrazca/telefona/poslovalnice,
- navesti datum/uro in kraj dogodka ter okoliščine.
