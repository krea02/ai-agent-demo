# Prometna nesreča – kaj je treba urediti
Če gre za prometno nesrečo, agent preveri:
- Ali so poškodovani ljudje? (če DA → klic 112/policija)
- Ali je bila poklicana policija? (zapisnik)
- Ali je izpolnjeno Evropsko poročilo o prometni nesreči? (če ni policije)

Za obravnavo škode je koristno pripraviti:
- fotografije kraja in poškodb,
- podatke druge stranke (ime, registracija, zavarovalnica),
- evropsko poročilo ali policijski zapisnik.

Nadaljnji koraki:
- zavarovalnica lahko zahteva ogled vozila ali dodatna dokazila,
- stranka sledi navodilom iz škodnega primera (številka primera).
