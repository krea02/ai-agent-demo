# Poškodba stekla
Za škodo na steklu agent zbere:
- katero steklo je poškodovano (vetrobransko/bočno/zadnje),
- ali je poškodba razpoka ali udarec,
- ali je vozilo varno za vožnjo.

Prijava:
- spletni obrazec/telefon/poslovalnica.
Stranka naj pripravi:
- številko police ali identifikacijske podatke,
- fotografijo poškodbe (če je možno).

Nadaljevanje:
- zavarovalnica poda navodila za popravilo ali menjavo.
