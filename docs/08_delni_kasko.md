# Delni kasko – pomen
Delni kasko je kritje za izbrane vrste škodnih dogodkov (ne nujno vse).
Tipični primeri kritij:
- stekla,
- toča,
- kraja,
- požar,
- živali (nalet na žival).

Natančen obseg je odvisen od paketa/izbranih kritij na polici.
Če stranka ne ve, katera kritja ima, agent predlaga preverbo police.
