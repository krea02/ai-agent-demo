# Kaj potrebujem za odškodnino (dokazi)
Za obravnavo odškodnine je priporočljivo pripraviti:
- fotografije škode,
- policijski zapisnik (če je bil opravljen),
- evropsko poročilo o prometni nesreči (če je relevantno),
- račune/predračune popravila (če obstajajo),
- podatke o pričah (če obstajajo).

Zavarovalnica lahko zahteva tudi ogled vozila ali dodatno dokumentacijo.
