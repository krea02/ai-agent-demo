# Roki pri prijavi škode
Škodo je priporočljivo prijaviti čim prej po dogodku.
Pri nekaterih primerih (npr. kraja) je obvezna takojšnja prijava policiji.

Če stranka ne pozna vseh podatkov:
- agent naj vseeno začne prijavo (osnovni podatki),
- stranka lahko dodatna dokazila dostavi naknadno po navodilih škodnega primera.
