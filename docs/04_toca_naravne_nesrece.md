# Toča / naravne nesreče
Če je škoda nastala zaradi toče ali druge naravne nesreče:
- priporočeno je fotografiranje vozila takoj, ko je varno,
- zabeležiti datum/uro in lokacijo.

Prijava zavarovalnici:
- spletni obrazec/telefon/poslovalnica.

Zavarovalnica lahko predlaga:
- ogled vozila,
- navodila glede popravila (pooblaščeni servisi ali cenitev).
